## 📌 Proyecto Java - Sistema de Clientes y Tickets

🧾 Descripción
- Este proyecto consiste en un sistema desarrollado en Java que permite gestionar clientes y tickets de soporte.
El sistema permite registrar clientes, crear tickets, cerrarlos y calificarlos, además de generar un resumen final del estado del sistema.

---

⚙️ Funcionalidades

* Registro y gestión de clientes
* Creación y administración de tickets
* Búsqueda de clientes por ID
* Eliminación y edición de clientes
* Cierre y calificación de tickets
* Persistencia de datos en archivos .txt
* Resumen final del sistema al cerrar

---

🛠️ Tecnologías utilizadas

* Java
* NetBeans IDE
* Programación Orientada a Objetos (POO)
* Manejo de archivos

---

📂 Estructura del proyecto

* src/ → Código fuente del sistema
* nbproject/ → Configuración de NetBeans
* archivos .txt → almacenamiento de datos

---

🚀 Cómo ejecutar

1. Descargar o clonar el repositorio.
2. Abrir el proyecto en NetBeans o cualquier IDE compatible con Java.
3. Ejecutar la clase **Main.java**.

Al iniciar el programa, se debe seleccionar el modo de uso:

* 1 → Consola
* 2 → Ventana gráfica (JOptionPane)

---

💾 Notas

* El sistema guarda automáticamente los datos en archivos .txt al salir.
* Se recomienda cerrar correctamente el programa para asegurar la persistencia de datos.
